This is a Traveling salesman algorithm 
it uses the lexicographic algorithm to generate
all the permutations based on the string of states provided in the first array
this array is then sent to a few functions that will save the tour and cost and print the resulting 
string and cost of the tour. 

Running this program:
If you are in linux use the provided make file

If you are in Windows use the provided run.bat batch file provided and it will hault the terminal before exiting.
